# WEARNWALK Inventory & Sales Tracker

Simple offline app for:

- Product inventory management
- Sales tracking
- Bill generation and printing with shop name **WEARNWALK**

## How to use

1. Open `index.html` in your browser.
2. Add products in the **Add / Update Product** section.
3. Create sales from **Create Sale & Generate Bill**.
4. A printable bill is generated automatically for each sale.

## Notes

- Data is saved in browser local storage, so it stays on your machine.
- Bill numbers auto-generate like `WW-00001`, `WW-00002`, etc.
